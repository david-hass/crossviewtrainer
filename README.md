# CrossViewTrainer

Minimal browser trainer for cross-view dot spotting.

## How it works
- Two circular views show the same rotating cone area.
- One side contains exactly one extra dot.
- Click the exact cell (ring + sector), then press `Solve`.
- Correct cell is highlighted; extra dot gets a white outline.

## Controls
- `New Round`: generate a new puzzle.
- `Solve`: check your selected cell and pause rotation.
- `Rotation Speed`: fine speed control.
